# Mini Web Synth

A browser-based synthesizer with a two-octave piano keyboard interface.

## Demo

[Live Demo](https://luana-l.github.io/amfm-keyboard/)

## Features

- Two-octave piano keyboard playable with mouse or computer keyboard
- Three synthesis modes: **Additive**, **AM**, and **FM**
- Waveform selection (sine, square, sawtooth, triangle)
- ADSR envelope (attack, decay, sustain, release)
- LFO with adjustable rate and depth (vibrato)
- Per-mode parameters (partials count, mod frequency, mod depth/index)
- 8-voice polyphony with master volume control
- Reactive background visualization that responds to notes played
